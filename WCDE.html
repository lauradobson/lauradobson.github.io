Waterloo Cases in Design Engineering: Website

Back

Goal

To develop a website for Waterloo Cases in Design Engineering (WCDE) that allows students to search for specific case studies and employees to add case studies to the database.

Approach

  1. Determine client's requirements
    • Who will be the users of the system?
    • What actions do you want performed on the system?
    • What do you want displayed by the system?
  2. Develop ER Diagram
    • Determine entities
    • Determine attributes and primary keys
    • Determine relationships, order of the relationships and foreign keys
  3. Implement objets in MySQL database
  4. Develop web based application
    • Supported by the database
    • Includes the following functionalities:
      • Searching (Name, First Name, Last Name, Key Words, Date, Topic)
      • Importing
      • Error Prevention

Tools Used

  • MySQL
  • PHP
  • HTML
  • CSS
  • JavaScript
