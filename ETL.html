ETL Process to the Cloud

Back

Goal

To extract, transform and load data from a client's source system (in-house ERP) to a Cloud ERP.

Approach

  1. Determine the client's requirements for the target system
  2. Define all business objects to be created within the client's source database
    • Staging Tables
    • Batch Sequence
    • Custom PL/SQL Package
  3. Develop the custom package
    • Clean staging tables
    • Extract data from source system and load into staging tables
    • Transform and Validate data
    • Export Data into flat files
  4. Load data into the target system

Tools Used

  • PL/SQL
  • Advanced SQL Querying
  • Oracle ERP Systems
