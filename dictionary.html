Counting the Unique Words

Back

Goal

To measure the time taken to extract all unique words from 25 text files and output the top 20 most reoccurring words using four different dictionaries.

Approach

  1. Develop methods for following dictionaries:
    • Unsorted Array
    • Sorted Array
    • Binary Search Tree
    • Hash Table
  2. Run each method and compare the results and run-time.

Tools Used

  • C#
